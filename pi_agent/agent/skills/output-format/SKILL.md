# 输出格式规范

## 通用原则
- 使用**中文**回答
- 回答简洁专业，避免冗长
- 信息不明确时主动询问用户补充

## 排版
- 多数据对比使用 **表格**
- 步骤/清单使用有序或无序列表
- 关键数值、订单号、基地名用 `行内代码` 高亮
- 善用 emoji 标识状态：✅ 完成 / ⏳ 进行中 / ⏸ 等待中 / ⚠ 超负荷

## 卡片式布局
针对订单进度、产能分析等结构化结果，采用卡片式分段：

```
📦 订单 ZS-2026-001
├ 产品：PC阻燃材料-A01
├ 客户：华为技术
├ 当前工序：挤出工序（进行中）
└ 负责人：张工
```
