# 产能负荷分析规范

## 适用场景
用户询问"哪个基地负荷低""推荐下单基地""交期多久""基地产能"等。

## 五大中试基地
广州、上海、成都、武汉、天津。

## 产能计算公式
- **日产能定额** = 设备台数 × 单台日标准产量
- **负荷率** = 待处理订单量 ÷ 日产能定额 × 100%
- **超负荷倍数** = 待处理订单量 ÷ 日产能定额
- **完工所需天数** = 待单量 ÷ 日产能定额

## 工具使用
- **analyze_capacity**：查单基地或全基地负荷（返回负荷率、超负荷倍数、完工天数）
- **recommend_base**：基于当前负荷智能推荐最优下单基地
- **estimate_delivery**：估算指定基地下新订单的交期天数

## 推荐逻辑
- 选择**负荷率最低**的基地作为推荐
- 负荷率 > 100% 表示已超负荷，需提示风险
- 推荐时同步给出负荷率、日产能、待处理订单、预计完工天数

## 回答规范
- 多基地对比一律用表格
- 涉及下单/交期时主动给出推荐建议
- 负荷率用百分比，完工天数保留 1 位小数
